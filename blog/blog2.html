Dutch Foods 'n Stuff, My Minecraft Mod

Posted on October 1, 2025

In this post, I'll walk you through how I created my own minecraft mod!

Dutch Foods 'n Stuff

Hello everyone! In this blog post, I want to share my experience making a Minecraft mod ("Dutch Foods 'n Stuff"). This mod adds Dutch foods and other items to the game. Here's how I made it.

Coding

To code my mod I used Intelij IDEA and Fabric 1.21.8, with java as the language. To get started I used the Fabric docs, youtube tutorials, and minecraft code examples. The mod adds several new food items, such as stroopwafels, oliebollen, bitterballen, and way more! Each food item has its own unique properties, such as hunger restoration and saturation levels.

In addition to food items, the mod also adds Dutch items, like klompen.
A very important thing you should not forget is to install the "Minecraft Development" plugin in Intelij IDEA, this makes coding a lot easier.

Textures

For the textures I created custom textures for each item using a pixel art program called Aseprite.
At first I just made them by hand but for later images I used reference images.
